# Virtual Machines Created in the Last 7 Days Dashboard

This repository provides dashboards and tools to monitor virtual machines created within the last 7 days in your VMware environment. These resources help track newly added virtual machines for better management and auditing.

## Features
- **New VM Tracking**: Identify virtual machines created within the last week.
- **Auditing and Reporting**: Gain insights into recent additions to your infrastructure.
- **Management Insights**: Ensure compliance with organizational policies for new virtual machines.

## Prerequisites
- **VMware vRealize Operations Manager**: Version 7.x or 8.x.
- **Permissions**: Ensure administrative access to import dashboards and views.

## Installation

### Import Dashboards
1. Navigate to `Dashboards` > `Actions` > `Manage Dashboard` > `Import`.
2. Select the relevant dashboard files from this repository.

### Import Views
1. Navigate to `Views` > `Import`.
2. Import all provided view files to enhance insights into newly created VMs.

## Usage
After importing the files, the dashboards and views will be accessible in vROps. Use them to:
- Track and audit virtual machines created within the last 7 days.
- Monitor compliance with policies for newly created resources.
- Manage and analyze trends in virtual machine provisioning.

## Repository Contents
- **Dashboards**: Pre-configured dashboards for tracking recent VM creations.
- **Views**: Custom views to provide detailed auditing and tracking capabilities.
